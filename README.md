# Mentorship Management System

## Project Description
This project is developed for the Database Management System course.

The Mentorship Management System helps students connect with mentors. Users can:

- View mentor details
- Add mentors
- Connect frontend with backend
- Store mentor data using MySQL database

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

---

## Features

- Add mentor information
- Display mentor list
- Store data in database
- REST API integration
- Frontend and backend connectivity

---
